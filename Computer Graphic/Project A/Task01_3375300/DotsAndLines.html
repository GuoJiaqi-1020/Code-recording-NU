This is my Small Project 1, whch is modified based on the file "DotsAndLines"

I added some vertices to the program, which can form a tetrahedron.
1. I have add 4 extra vertices to give this 3D object a basic skeleton.
2. I have add some rotation to make the 3D object more visible.
3. The 3D model has been reduced to 0.4 times its original size, Using "Translation matrix".
4. Besides, I have add two button for controling the rotating speed of 3D obj.
5. We also added a different color for each vertex in order to make the graphics beautiful.
